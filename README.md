# ♻️ AI-Driven Smart Waste Segregation System

An intelligent full-stack waste management platform that combines **Computer Vision**, **Machine Learning**, **IoT**, and **Generative AI** to automate waste segregation, provide real-time monitoring, and educate users about sustainable waste disposal.

## 🚀 Overview

The Smart Waste Segregation System uses deep learning models to classify different types of waste and automatically segregate them using IoT hardware. The platform also includes an AI-powered chatbot built using Retrieval-Augmented Generation (RAG) to answer waste management and recycling-related queries.

This project was developed as part of **HackGenX 2026**, where it was selected among the **Top 70 teams out of 700+ teams nationwide**.

---

# ✨ Features

- 🧠 AI-powered waste classification using TensorFlow
- 📷 Real-time image-based waste detection
- 📊 Interactive dashboard for monitoring predictions
- 🤖 RAG-powered AI chatbot for waste segregation guidance
- 🌱 Recycling and sustainability recommendations
- 📡 ESP32-based IoT integration
- 📈 Live sensor monitoring and analytics
- 📱 Responsive web interface built with React
- 🔄 REST API-based communication between frontend and backend
